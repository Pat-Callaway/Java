### Terms
1. Scanning
- Also known as "lexing" (lexical analysis)—Linear stream of characters that get "chunked" together to form tokens.
2. Tokens
- Can be single characters, string literals, identifiers, etc.
3. Parsing
- This is where the syntax gets grammar.
- A parser takes a flat stream of tokens and creates a tree structure that mirrors the nested nature of grammar.
4. Syntax Tree
- Parse Tree or Abstract Syntax Tree (AST)
5. Static Analysis 
- First is binding or resolution
- For each identifier we find out where the name(s) are defined and wire them together
